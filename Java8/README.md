Java 8
=======
This project contains sample code (along with easy to understand comments) to learn Java 8 features and enhancements.

How to use?
-----------
Just import this project in Eclipse as an existing project, do necessary configuration to fix build path errors (e.g. setup JRE 1.8 Libraries).


New Language Features in Java 8
--------------------------------
* [Lambda Expressions/Closures] (https://github.com/tirthalpatel/Learning-Java/tree/master/Java8/src/com/tirthal/learning/langfeatures/lambda)
* Default and Static Methods in Interfaces (https://github.com/tirthalpatel/Learning-Java/blob/master/Java8/src/com/tirthal/learning/langfeatures/Ex_DefaultStaticMethodsInInterface.java)
* Method References
* Improved Type Inference in Generics
* Repeating Annotations
* Type Annotations
* Method Parameter Reflection


Libraries Enhancements
----------------------
* Collection framework enhancements & Stream API	
* Date and Time API
* The java.lang and java.util packages enhancements
* Nashorn JavaScript Engine
* Introduction to Java FX 8
